## k3sup

### 组建单控制节点

```shell
# 需本机已安装 k3sup（存档步骤，当前主路径是 ansible）

# --embedded-registry
# ref: https://docs.k3s.io/installation/registry-mirror?_highlight=spegel
# --k3s-version 指定版本 v1.31.9+k3s1 v1.33.2+k3s1
# ref: https://github.com/k3s-io/k3s/releases
# --disable local-storage
# 使用openebs的localpv

export SSH_USER="<SSH_USER>"
export SSH_PRIVATE_KEY_PATH="<SSH_PRIVATE_KEY_PATH>"

# 主节点
k3sup install --user "$SSH_USER" --ip 192.168.6.80 \
  --k3s-extra-args "--disable traefik --disable servicelb --flannel-backend=none --disable-network-policy --disable-kube-proxy --disable local-storage --disable coredns --disable metrics-server" \
  --k3s-version v1.33.2+k3s1 \
  --ssh-key "$SSH_PRIVATE_KEY_PATH"

# 加入 agent（worker）
k3sup join \
    --user "$SSH_USER" \
    --ip 192.168.6.111 \
    --server-ip 192.168.6.80 \
    --server-user "$SSH_USER" \
    --k3s-version v1.33.2+k3s1 \
    --ssh-key "$SSH_PRIVATE_KEY_PATH"

# free-network (optional)
ansible-playbook ansible/playbooks/change-k3s-proxy.yml

```

### 卸载说明

如果需要卸载 k3s，需要在每个节点上执行以下操作：

1. 在工作节点（agent）上执行：

```shell
# 对于 agent 节点
/usr/local/bin/k3s-agent-uninstall.sh
ansible-playbook ansible/playbooks/delete-k3s.yml
```

1. 在主节点（server）上执行：

```shell
# 对于 server 节点
/usr/local/bin/k3s-uninstall.sh
ansible-playbook ansible/playbooks/delete-k3s.yml
```

注意：

- 卸载脚本会停止所有 k3s 进程
- 删除 k3s 相关的文件和目录
- 删除 iptables 规则和 ipvs 表
- 移除 k3s 创建的网络接口
